C and C++ parser for CLI arguments.

Features:

- [x] Support for short and long arguments
- [x] Support for grouped short switches(`-aBcD` is the same as `-a -B -c -D`)
- [x] Support for arguments with variable assignment(`--test=test` or `--test test`)
- [x] Support for default arguments
- [x] Support for array types
- [x] Complete C and C++ APIs
